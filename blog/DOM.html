DOM

1.What the DOM is and how it relates to CSS and HTML?

The DOM (Document Object Model) serves as the interface between HTML, CSS, and JavaScript. It represents the structure of an HTML document as a tree of objects, allowing JavaScript to interact with and manipulate the content and style of a webpage dynamically. CSS styles are applied to HTML elements through the DOM, enabling the presentation of the webpage to be styled according to the specified CSS rules. Dom Hierarchy

2.How is the DOM structured?

The DOM represents the HTML document as a hierarchical tree structure, where each node represents an element, attribute, or text within the document.
NODE: In the DOM, a "node" is an individual object representing an element, attribute, or text in an HTML document, such as <div> <p> <a>. These nodes form a tree-like structure, allowing developers to interact with and manipulate the document's content and structure dynamically.

3.How can you access & manipulate elements in the DOM?

You can access elements in the DOM using various methods, such as getElementById, getElementsByClassName, querySelector, and querySelectorAll. And manipulate them by changing their attributes, adding or removing classes, modifying their text content, or altering their structure.

4.What are DOM events?

DOM events are actions that happen in the HTML document or browser window, such as clicks, mouse movements, keypresses, page load, etc.
